FastAPI + PostgreSQL

Minimalni UI za autentikaciju

Ovaj dio pokriva register, login, spremanje JWT tokena, dohvat trenutnog korisnika i logout. Ostali članovi tima mogu kasnije nadograditi termine, prijave i admin funkcionalnosti. 

Klikni gumb za provjeru API-ja.

Frontend script se učitava...

Register

Register stvara običnog studenta. Admin/profesor korisnici se dodaju kroz seed podatke ili admin funkcionalnost.

Login

Login koristi OAuth2 password formu. U polje username šaljemo email, isto kao u Swagger Authorize prozoru.

Trenutni korisnik

Ako je token spremljen u localStorage, ovdje se prikazuje rezultat endpointa GET /auth/me.

Niste prijavljeni.

Termini

Ovaj prikaz je samo priprema za dio osobe koja radi termine. Ako endpoint GET /termini postoji, prikazat će se osnovni JSON odgovor. 

Termini još nisu učitani.

Admin panel

Admin UI za kreiranje i uređivanje termina ide kasnije u dio osobe zadužene za CRUD termina. Auth dio ovdje samo omogućuje da se zna tko je prijavljen i koja mu je rola.
